What Is It?

Omega is the combined frontend and backend running the gaming experience.  The EHA systems run emulation for various consoles and arcades.  The frontend experience runs Launchbox.  The backend or the individual emulators running various consoles, portables, and arcade games all use different emulators.  Previous versions of the drives ran Hyperspin as its frontend with Rocketlauncher as its backend.  Launchbox is actively being developed, enhanced, and supported by the community and requires a $75 license to run in what's called Bigbox mode.  It's the pretty frontend everyone primarily uses.

MSVCP140.dll error when playing/opening xxx game?

msvcp140.dll is a VCRUNTIME DLL file, reinstalling the visual c redistributable files again should fix this.
Download them here:  https://www.techpowerup.com/download/visual-c-redistributable-runtime-package-all-in-one/ and run the .bat installer to install all of them.

 

How do you save a game to favorites on omega?
When you have a game highlighted, press the player one coin button. That will bring up a new menu. Click on the favorite option there. 
Note, that it won't show up on your favorites until you close out the Big box software and relaunch it.

 

How can I use buttons or controller buttons for ps1 games?
Ps1 is mapped to the control panel by default. If you want to use controllers, map them how you want them in the emulator.

How do I disable LEDBlinky & OBS Studio if I don't use them?

Open up task manager (CTRL + ESC) and check the Startup folder for Ledblinky and OBS Studio.
Right-click the item and click disable.

 

Teknoparrot says there's an update, should I update?
Be wary, Tekno is not stable in any way, and unfortunately, some updates break things.
Do so at your own risk, updates can require new versions of games which can be hard to get. 
If you must update Teknoparrot make it easier on yourself and backup the entire folder first somewhere else. 
The average user that does not install their own emulators should never update.

 

Game XYZ got removed from the wheel after I favorited it, where'd it go?
It doesn't remove them, but it will move them to the start of the wheel depending on your settings.
Go to options->views and make sure "show favorited games first" is unchecked

Omega Quick Start Guide

This Quick Start Guide is for fast learners, and  people who do not like user’s manuals and just want to dive in)!    

To receive your machine from the shipping company, unpack it, and move it into your residence, please see those sections later in this manual.  This Quick Start Guide presumes you have your machine in a safe location, have plugged it in and the machine has electrical power. 

 

NOTE:  EHA supports several different button sequences depending on your machine:

  • 2 player EHA machines and older 4-player EHA machines - [Shift] Key is Player 1's Start Button.
  • KEYWIZ - If your EHA machine has a KEYWIZ your [Shift] key on your joystick panel is the “Shazaam” button.
  • Newer (after 6/2022) EHA 4-player machines (IPAC4) - Your [Shift] key is Player 3's 7th button (i.e., Player 3 - 4th button from the left top).

 

  1. Turning On Your Machine:
  • Uprights (MegaCade, Classic, Stealth) – The power button is located on top of the machine (upper left or right top of machine). It is a standard arcade push button (typically black). Push it, and it will turn on your machine.
  • Tabletops – The power button is located on the back center portion of the cabinet.
  • Pedestals – The power button is located on the back of the machine, near the center of the pedestal cabinet, opposite the HDMI port.

 

2.  Loading a Game:

  • After you turn on your machine, an introduction video will automatically load. Allow the video to run for 30 seconds (this allows the system to fully load).   After 30 seconds, push any button or push any position on any joystick to skip the introduction video. You will be at the Big Box main wheel.
  • You can move down the Big Box wheel by pressing Player 1's Joystick down. Alternatively, you can move up the Big Box wheel by pressing the Player 1's Joystick up.
  • Use Player 1's Joystick to go to ARCADE CLASSICS on the Big Box main wheel.
  • Press Player 1’s START Button (a white or black man icon button) to select ARCADE CLASSICS.   You will then be in the Arcade Classics Wheel (menu).
  • Load any game in ARCADE CLASSICS by pressing Player 1’s START Button.  For example, you could push Player 1’s Joystick up or down to navigate to the game “Batman Forever,” then press Player 1’s START Button to load the game.  Alternatively, you can press Player 1’s Joystick left or right to scroll through the alphabet to the letter B and then press Player 1’s Joystick down until you come to Batman Forever (and then press Player 1’s START Button to load the game).
  • Wait for the game to load.  When the game is fully loaded (this could take a few minutes), credit the game by pressing the Credit Button (also called the coin button, typically it looks like a dollar sign - $. ).  You can do this multiple times.  If you loaded Batman Forever, push the Credit Button four times.  This also works for Players 2, 3 and 4’s credit button.
  • Press 1 (2, 3, or 4) Player’s START Button to start a 1 player or multiplayer game.   If you loaded Batman Forever, press Player 1’s START Button and then press Player 2’s START Button to select a two-player game.  Press Player 1 and 2’s START Buttons again to start the game.
  • To exit any game at any time Press and HOLD Player 1’s START Button/OR your [SHIFT] Button and simultaneously push Player 2’s START Button.  (Note: do not try at first at least to push them at the same time, first HOLD Player’s Start Button then simultaneously Press Player 2’s START Button).   For example, if you played Batman Forever, Press and HOLD Player 1’s START Button and simultaneously push Player 2’s START Button.   This will take you back to Arcade Classics wheel.   To go back to the Big Box Main Wheel Press and HOLD Player 1’s START Button and simultaneously push Player 2’s START Button.   Note: In the Big Box menu system press Player 1’s START Button to go forward in the menus and press and HOLD Player 1’s START button/OR your [SHIFT] Button and simultaneously push Player 2’s START button to go backwards in the Big Box menu system.

 

3. Turning Off the Machine

  • When you are ready to turn off your machine, go to the Big Box Main Wheel (by Holding Player 1’s START button/OR your [SHIFT] Button and simultaneously push Player 2’s START button). Then, once again, Hold Player 1’s START button/OR your [SHIFT] Button and simultaneously push Player 2’s START button.  This will take you to the “Big Box Unified” screen.  Use Player 1’s Joystick to point to Exit and then press Player 1’s START Button to select it.  This will exit you out of Big Box and take you to the Windows Desktop.
  •  Now, use your trackball as a mouse and point to the Windows Start Button.  There is a silver auxiliary toggle switch above Player 1’s Start Button. Make sure the auxiliary toggle switch is positioned to the right.  There are three horizontal buttons above the trackball.  These operate as your left, middle and right mouse buttons.  While pointing with the trackball to the Windows Start Button, click the left mouse button.  Then, click on the “Power” icon and then click on “Shutdown."   
  •  Alternatively, you can hold the power button (what you used to turn the machine on with) to turn the machine off.

 

4. Quick Big Box Interface/Controls Summary (see also Appendix 1)

  • Moving Forward in Menus and Games - Press Player 1’s Joystick UP and DOWN to scroll through the wheels and/or game titles.
  • Selecting a Wheel or Game - Press Player 1’s START button to select a wheel from the main wheel or to select and play a game.
  • Moving Backward in a Wheel/Menu - HOLD Player 1’s START Button/OR your [SHIFT] Button and then press Player 2’s START button to go backward in the menu system.
  •  Quitting a Game and Going Back to a Wheel or Menu - HOLD Player 1’s START Button/OR your [SHIFT] Button and then press Player 2’s START button to stop a game while you are playing it and go back to the wheel or menu.
  • Searching by a Letter for Game Titles – At a game menu/wheel, more Player 1’s Joystick LEFT or RIGHT to search for game titles.  Just move the cursor to the letter you want to search for.
  • Search for a Game Title in a Wheel – In a game menu (i.e. Arcade Classics) Press Player 1’s Joystick LEFT or RIGHT to get to bring up the alphabet search.    Instead of clicking on a letter, click on the first icon which is a magnifying glass (search feature). An on-screen keyboard is displayed.  You can enter the name of a game to search on here using Player 1’s Joystick or the keyboard.  Use Player 1’s START Button to enter the search. Enter “ESC” on the keyboard to back out of the results to the main wheel.
  • Global Search – Press Player 2’s CREDIT (COIN $) Button to bring up the Global Search. This will search all games on Omega.   You can enter the name of a game to search on using Player 1’s Joystick (and Player 1’s START button to select letters) or use a keyboard.  After you have entered the title to search on press Player1’s START Button to start the search (i.e. “Done”).  A wheel of just your search results is created. Press Player 1’s Joystick to navigate the wheel and press Player’s 1’s START button to load a game. You can HOLD Player 1’s START Button/OR your [SHIFT] Button and then press Player 2’s START button to back out of the search results or you can press “ESC” on the keyboard to back out as well.  Alternatively, HOLD Player 1’s START Button/OR your [SHIFT] Button and then press Player 2’s START button multiple times until you get to the “Big Box Unified” screen (where you can exit Big Box).  Use Player 1’s Joystick to navigate to “Search” and then press Player1’s Start Button.  Enter your search criteria and Player1’s Start Button to enter it.
  • Game Pause/Reset - (This works on ALL games in LaunchBox)– While playing a game, HOLD Player 1 START Button/OR your [SHIFT] Button and Press Player 1 Joystick Down - This will pause the emulator and give you several options including, viewing the game’s manual, resetting the game, resuming the game, saving/loading a game and exiting the game, etc.  Use Player 1’s Joystick and Player 1’s START button to navigate and select options.
  • Changing Input Control (Arcade Classics - MAME games ONLY) -  HOLD Player 1 START Button/OR your [SHIFT] Button and Press Player 1 Joystick Up. This allows you to change “Input This Machine” and set your control devices.   Press this same command again to unpause the game.  See later in this manual for additional information.
